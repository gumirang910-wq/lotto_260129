# 뉴스 검색 챗봇

**키워드 검색**을 통해 **관련 뉴스 10개**를 검색해 주는 챗봇입니다.

---

## 한 번에 실행 (권장)

1. **`챗봇_실행.bat`** 을 더블클릭합니다.
2. 최초 실행 시 의존성 설치 후 서버가 시작되고, 브라우저가 **http://localhost:3000** 으로 열립니다.
3. 검색창에 **키워드**를 입력하면 관련 뉴스 10개가 표시됩니다.

(npm이 `offline=true` 로 설정된 PC에서는 먼저 터미널에서 `npm config set offline false` 실행 후 배치 파일을 실행하세요.)

---

## 실행 방법 (Node.js · 터미널)

1. **의존성 설치**
   ```bash
   npm install
   ```

2. **챗봇 실행**
   ```bash
   npm start
   ```

3. 브라우저에서 **http://localhost:3000** 을 열고, 검색창에 **키워드**를 입력하면 관련 뉴스 10개가 채팅 형태로 표시됩니다.

---

## 설치 없이 사용 (HTML만)

- **`standalone.html`** 을 더블클릭해 브라우저에서 엽니다.
- 키워드 입력 후 **검색** 버튼을 누르면 **구글 뉴스** 검색 페이지가 새 탭에서 열립니다.

---

## 구성

| 파일 | 설명 |
|------|------|
| `server.js` | Express 서버 + 뉴스 검색 API |
| `public/index.html` | 챗봇 UI (키워드 입력 → 뉴스 10개 표시) |
| `package.json` | Node 의존성 (express, duck-duck-scrape) |

뉴스 검색에는 **DuckDuckGo**를 사용합니다. (API 키 불필요)
